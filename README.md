# Aplicativo

A tela inicial é a tela responsável por buscar os próximos filmes a estrear. Ao selecionar um filme são exibidos detalhes do filme como sinópse, gênero, descrição, capa e data de lançamento.
A lista de filmes fica guardada no aplicativo e pode ser acessada de modo offline.
